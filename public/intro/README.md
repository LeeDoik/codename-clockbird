# 오프닝 시네마틱 에셋 (HEART OF STEEL)

`IntroScene`(`src/client/scenes/IntroScene.js`)이 이 폴더의 파일을 **런타임에 `/intro/...` 로 로드**한다.
파일이 없어도 게임은 돌아간다 — 그림은 어두운 배경으로, 소리는 무음으로 폴백한다.
그래서 아트를 붙이는 순서와 무관하게 언제든 타이밍·대사를 확인할 수 있고, 아래 이름대로
파일만 떨구면 코드 수정 없이 그대로 붙는다.

## 패널 이미지 (필수 아님, 있으면 표시)

스토리보드 8개 아트 컷. 캔버스(896×576, 약 14:9)에 **비율 유지 커버 맞춤**되고 느린 줌이 걸린다.
가로로 넉넉한 그림(16:9 이상)이 잘 어울린다. PNG 권장.

| 파일 | 스토리보드 컷 | 내용 |
|---|---|---|
| `01.png` | 1 | 증기 도시의 평화로운 전경 |
| `02.png` | 2 | 철컹거리며 거리를 습격하는 강철 로봇 군단 |
| `03.png` | 3 | 사람들을 학살하는 강철 로봇들 |
| `04.png` | 4 | 로봇 군단을 거느린 브루주아 / 그늘로 피하는 사람들 |
| `05.png` | 5 | 서로 다른 장소의 레지스탕스들 (쪽지·눈짓) |
| `06.png` | 6 | 각자의 장소에서 은밀히 연결되는 레지스탕스 |
| `07.png` | 7 | 표적 — 고철덩어리와 그 주인 |
| `08.png` | 8 | "우리는…" (손·엠블럼 클로즈업 등) |

> 9·10컷(타이틀 "HEART OF STEEL" + 문구)은 **엔진에서 텍스트로 그리므로 이미지가 필요 없다.**

## 선택 에셋

| 파일 | 쓰임 |
|---|---|
| `music.mp3` | 오프닝 배경 음악 (루프, 볼륨 0.45) |
| `steam.mp3` | 1컷 증기 소리 |
| `clang.mp3` | 9컷 타이틀 등장 금속음 |
| `handler.png` | 브릿지(튜토리얼 전환) 접선책 초상. 없으면 대사만 중앙 정렬 |

## 참고

- 타이밍·대사는 `IntroScene.js` 상단 `BEATS` 배열에 있다. 초 구간을 바꾸려면 각 컷의 `dur`(ms)를 고친다.
- 오프닝은 **[Space]/[Esc]/클릭으로 즉시 스킵**된다. 개발 중 아예 건너뛰려면 URL 에 `?nointro`.
- 파일 이름·경로는 위와 정확히 일치해야 한다 (`/intro/01.png` …). 대소문자·확장자 주의.
