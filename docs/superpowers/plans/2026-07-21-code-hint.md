# 접선 코드 힌트 (글자 수·카테고리 공개) 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 스테이지 시작 시점부터 접선 코드의 글자 수·카테고리를 플레이어에게 공개해 추리 범위를 좁힌다.

**Architecture:** 서버 `toClientView` 에 `hint: { length, category }` 필드를 추가하고(비유출 정책의 의도적 완화), 클라이언트는 시작 브리핑 쪽지와 단서 수첩 두 곳에 표시한다. wordGen·중복 판정·경계 밸런스는 건드리지 않는다.

**Tech Stack:** Node 22 + Express 5 (서버) / Phaser 3 (클라이언트) / 스모크 스크립트 (`npm run smoke`, 실서버 대상 통합 테스트 — 이 리포에 단위 테스트 프레임워크는 없다)

**스펙:** `docs/superpowers/specs/2026-07-21-code-hint-design.md`

## Global Constraints

- 튜토리얼은 손대지 않는다 — `TutorialScene.js` · `tutorialSession.js` · `routes/tutorial.js` 수정 금지 (`toClientView` 는 stage 전용임을 확인함)
- 코드 입력 검증(글자 수 차단·경고)을 추가하지 않는다 — 힌트는 정보로만
- `codeWord` / `reason` 비유출은 유지 — 기존 smoke 검사가 그대로 통과해야 한다
- 필드명·형식 고정: `hint: { length: <정수>, category: <codewords.json 분류 키> }`
- 표기 고정 — 브리핑: `코드는 3글자 — 기계 쪽 단어다.` / 수첩: `접선 코드: ○○○ (기계)` (○ 는 `'○'.repeat(hint.length)`)
- 클라이언트는 `this.state.hint` 부재를 옵셔널 체이닝/가드로 방어 (없으면 힌트 줄 생략)
- 커밋 메시지는 리포 관례(한국어, `feat:`/`test:` 접두) + `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>` 트레일러

**smoke 실행 전제:** dev 서버가 떠 있어야 한다 (`.env` 의 `ANTHROPIC_API_KEY` 사용).
별도 터미널(또는 백그라운드)에서 `npm run dev:server` 를 먼저 실행해 둔다.
`node --watch` 라 서버 파일을 고치면 자동 재시작된다 — smoke 를 다시 띄우기만 하면 된다.
smoke 1회는 LLM 생성(연상 단어 5인 + 판정)을 실제로 태워 약 1~2분 걸린다.

- [ ] **Step 2: `#refreshClues` 에 힌트 머리줄 추가**

메서드 전체(기존 238-249행)를 다음으로 교체:

```js
  #refreshClues() {
    // 힌트는 단서 유무와 무관한 고정 머리줄 — 빈 수첩에서도 보인다.
    const hint = this.state.hint;
    const head = hint ? `접선 코드: ${'○'.repeat(hint.length)} (${hint.category})\n\n` : '';
    if (this.clues.size === 0) {
      this.clueText.setText(`${head}아직 수집한 단서가 없다.\n\n동료 근처에서 [F] 로 접선하면,\n그가 흘린 연상 단어가 여기 기록된다.`);
      return;
    }
    const lines = [];
    for (const { name, role, word, rescued } of this.clues.values()) {
      lines.push(`· ${name} (${role})\n     「${word}」${rescued ? '   ← 둘이 겹쳐 낸 단어' : ''}`);
    }
    lines.push(`\n수집한 단서 ${this.clues.size}개 — 이 단어들로 접선 코드를 추리하라.`);
    this.clueText.setText(head + lines.join('\n'));
  }
```
